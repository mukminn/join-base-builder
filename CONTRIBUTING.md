# Contributing to Base Builders December

Terima kasih atas minat Anda untuk berkontribusi! Proyek ini dibuat untuk Base Builders December program.

## Cara Berkontribusi

### 1. Fork Repository

Fork repository ini ke GitHub account Anda.

### 2. Clone Repository

```bash
git clone https://github.com/your-username/base-builders-december.git
cd base-builders-december
```

### 3. Create Branch

Buat branch baru untuk feature atau fix:

```bash
git checkout -b feature/your-feature-name
# atau
git checkout -b fix/your-fix-name
```

### 4. Make Changes

- Buat perubahan yang meaningful
- Follow code style yang sudah ada
- Add comments untuk complex logic
- Update documentation jika perlu

### 5. Commit Changes

Buat commit dengan message yang jelas:

```bash
git add .
git commit -m "Add: description of your changes"
```

**Commit Message Format:**
- `Add:` untuk new features
- `Fix:` untuk bug fixes
- `Update:` untuk updates
- `Docs:` untuk documentation
- `Refactor:` untuk code refactoring

### 6. Push dan Create Pull Request

```bash
git push origin feature/your-feature-name
```

Kemudian buat Pull Request di GitHub dengan description yang jelas.

## Code Style

- Use consistent indentation (2 spaces)
- Follow Solidity style guide untuk contracts
- Use meaningful variable names
- Add comments untuk complex logic
