JIKA TIDAK BISA:
========================================

A. Cek Node.js terinstall:
   - Buka CMD/PowerShell
   - Ketik: node --version
   - Harus muncul versi (contoh: v24.12.0)

B. Jika port 9000 sudah digunakan:
   - Edit file: mini-app\server.js
   - Ganti: const PORT = 9000;
   - Jadi: const PORT = 9001;
   - Save, lalu jalankan START-HERE.bat lagi

C. Clear browser cache:
   - Tekan Ctrl+Shift+Delete
   - Clear cache
   - Refresh page
