## ğŸ“ Smart Contracts

### 1. FeeGenerator.sol

Contract yang menghasilkan fees melalui deposit dan withdraw operations.

**Features:**
- Deposit ETH dengan fee 2%
- Withdraw ETH dengan fee 2%
- Track total fees generated
- User balance management

**Deploy:**
```bash
npm run deploy:base-sepolia  # Testnet
npm run deploy:base          # Mainnet
```

### 2. SimpleNFT.sol

NFT contract yang menghasilkan fees melalui mint operations.

**Features:**
- Mint NFT dengan fee 0.001 ETH
- Batch mint support
- Custom token URI
- Fee tracking
